# 📁 Uncategorized directory
This directory contains a bunch of important modules for the system. Due to them not fitting in one particular category, they are all here instead. I suppose this is a bit of a category itself… \
It contains:
- Generic (non device-specific) boot options and configuration;
- The Linux Kernel version to use;
- The Plymouth boot splash screen;
- Kmscon as a getty replacement;
- Configuration for fully open-source and closed NVIDIA GPU drivers, for 1630 and above only.
