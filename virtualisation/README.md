# 📁 Virtualisation directory
This directory contains modules for host virtualisation. \
They are the following:
- [Docker](https://www.docker.com/);
- [libvirt](https://libvirt.org/) with [Virt-Manager](https://virt-manager.org/) (and working NVIDIA GPU 3D acceleration!);
- [Waydroid](https://github.com/waydroid/waydroid).

VirtualBox, VMWare, and others are not here, for I do not use them, and thus cannot test that they work as intended.
